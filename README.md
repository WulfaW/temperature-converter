# Temperature Unit Converter

A simple and user-friendly web application to convert temperatures between Celsius, Fahrenheit, and Kelvin units. Perfect for quick conversions and educational purposes.

## Live Demo

Check out the working app here: [Temperature Unit Converter](https://glowing-blancmange-fd6a22.netlify.app/)

## Features

- Convert temperature values seamlessly between Celsius, Fahrenheit, and Kelvin
- Clean and intuitive user interface
- Responsive design for all devices
- Fast and accurate calculations

## How to Use

1. Enter the temperature value you want to convert.
2. Select the unit of the input temperature.
3. Select the desired output unit.
4. View the converted temperature instantly.

## Technologies Used

- HTML5
- CSS3
- JavaScript
